# memcard-site

Public site for the [MemCard](https://apps.apple.com/) iOS app — hosts the Privacy Policy and Support pages required for App Store submission.

Served via GitHub Pages from the root of `main`, custom domain `memcard.app`.

- Site: <https://memcard.app/>
- Privacy Policy: <https://memcard.app/privacy.html>
- Support: <https://memcard.app/support.html>
